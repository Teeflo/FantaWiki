# 🥤 FantaWiki — The Ultimate Fanta Flavor Encyclopedia

[![Vercel Deployment](https://img.shields.io/badge/Deploy-Vercel-black?style=for-the-badge&logo=vercel)](https://fantawiki.vercel.app)
[![React](https://img.shields.io/badge/React-19-blue?style=for-the-badge&logo=react)](https://react.dev)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-4.0-38B2AC?style=for-the-badge&logo=tailwind-css)](https://tailwindcss.com)

**FantaWiki** is an unofficial fan project dedicated to celebrating the incredible diversity of Fanta flavors around the globe. From classic Orange to mysterious limited editions, explore a rich catalog of over 85 varieties.

## 🎨 Visual Identity (2023 Rebrand)

The site embraces the new Fanta global Brand Identity launched in 2023:
- **Neo-brutalist / Pop-Art Style**: Thick borders, solid shadows, and vibrant colors.
- **"Paper Cut" Typography**: Featuring *Luckiest Guy* for a bold, hand-cut look.
- **Dynamic FantaCan Component**: A custom 3D-styled React component that adapts its color and label for every single flavor.

## 🚀 Features

- 🔍 **Full Catalog**: Search and filter by category, region, and status (Zero, Limited Edition).
- 🌍 **World Tour**: Explore flavors grouped by countries and regions (Japan, Brazil, Europe...).
- 🥤 **Flavor Details**: Dedicated pages with custom visual vibes for each taste.
- 📱 **Fully Responsive**: A fizzy experience on all screens.

## 🛠️ Tech Stack

- **Framework**: [Vite](https://vitejs.dev/) + [React 19](https://reactjs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router 7](https://reactrouter.com/)
- **Deployment**: [Vercel](https://vercel.com/)

## 📦 Local Installation

```bash
# Clone the repository
git clone https://github.com/Teeflo/FantaWiki.git

# Install dependencies
npm install

# Run the development server
npm run dev
```
